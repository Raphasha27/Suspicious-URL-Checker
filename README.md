## Overview

Advanced URL analysis tool that detects phishing attempts through URL parsing, homograph (lookalike character) detection, domain reputation simulation, and multi-factor risk scoring. Essential for security awareness and email filtering.

## Features

- **URL Parsing** — In-depth URL structure analysis
- **Homograph Detection** — Identifies lookalike characters and punycode abuse
- **Domain Reputation** — Simulated reputation scoring
- **Risk Scoring** — Weighted multi-factor risk assessment
- **Batch Processing** — Analyze multiple URLs from file input
- **Report Generation** — Detailed analysis reports in JSON/CSV

## Quick Start

```bash
git clone https://github.com/Raphasha27/Suspicious-URL-Checker.git
cd Suspicious-URL-Checker
pip install -r requirements.txt
python check.py --url "https://example.com"
```
